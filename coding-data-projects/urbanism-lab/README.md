# Urbanism Lab: Old Chicago Post Office Analysis

This folder contains coding, and data analysis materials from my Urbanism Lab research on the Old Chicago Post Office redevelopment.

The project examines neighborhood and property-value changes around the Old Chicago Post Office using parcel-level assessment data, ACS indicators, GIS methods, and economic benefit analysis. The broader research question is how the redevelopment of a major historic commercial asset affected nearby property values, land use patterns, and neighborhood change.

## Materials Included

- `property_values.rmd`: R Markdown files documenting the data cleaning, parcel-year panel construction, inflation adjustment, value-per-square-foot calculations, and figure-generation workflow.
- `property_values_code&output.pdf`: Knitted PDF outputs showing rendered analysis, code output, tables, and figures.

## Project Summary

This project uses historical parcel assessment data and spatially defined treatment/control areas around the Old Chicago Post Office to examine changes in assessed value over time. The analysis adjusts values for inflation, normalizes assessed value by building square footage, and compares trends across geographic rings surrounding the redevelopment site.

The project is part of a broader historic preservation and urban economic development analysis using PlaceEconomics framework.
