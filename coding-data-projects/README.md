# Coding and Data Projects

This folder contains coding and data analysis projects using R, Excel, GIS, and related tools. The projects demonstrate applied data cleaning, analysis, visualization, and research documentation for policy-relevant questions.

## Projects Included

### Urbanism Lab

The Urbanism Lab project examines the redevelopment of the Old Chicago Post Office and its surrounding neighborhood effects. The analysis uses parcel-level assessment data, ACS indicators, GIS methods, and economic benefit frameworks to study property-value change and neighborhood transformation.

## Skills Demonstrated

- R and R Markdown workflows
- Data cleaning and transformation
- Parcel-level data analysis
- Inflation adjustment
- Value-per-square-foot calculations
- Figure and table generation
- GIS-informed policy analysis
- Reproducible research documentation

## Data Note

Some raw datasets are not included because they are large, restricted, or stored locally. Code files and rendered outputs are included where possible to document the analysis workflow.
