# CRDC Discipline Analysis

The project analyzes disciplinary actions for K–12 students across Illinois using Civil Rights Data Collection (CRDC) data. The analysis focuses on in-school suspensions, out-of-school suspensions, expulsions, and corporal punishment.

## Materials Included

- `CRDC_Presentation.pptx`: Final presentation summarizing discipline trends, disparities, and visualizations.
- `Corporal Punishment-Illinois.xlsx`: Excel analysis spreadsheet for corporal punishment trends in Illinois.
- `Expulsions.xlsx`: Excel analysis spreadsheet for expulsion trends.
- `In-School Suspensions-Illinois.xlsx`: Excel analysis spreadsheet for in-school suspension trends in Illinois.
- `Out of School Suspensions-Illinois.xlsx`: Excel analysis spreadsheet for out-of-school suspension trends in Illinois.

## Project Summary

For this project, I reviewed federal CRDC reporting guidelines and analyzed Illinois school discipline data across multiple school years. I organized CRDC data in Excel, calculated discipline trends, and prepared charts comparing disciplinary actions across student groups.

The analysis examines how disciplinary outcomes vary by race/ethnicity, gender, disability status, English Learner status, locale, and school type. The final presentation communicates key patterns in exclusionary discipline and highlights persistent disparities in school discipline practices.
