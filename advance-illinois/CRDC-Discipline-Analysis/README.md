# CRDC Discipline Analysis

This project analyzes disciplinary actions for K–12 students across Illinois using Civil Rights Data Collection (CRDC) data.

## Materials Included

- `analysis-spreadsheets/`: Excel workbooks used to organize CRDC data, calculate discipline trends, and prepare charts
- `documentation/`: CRDC data definitions and reporting notes
- `presentation/`: Final slide deck summarizing key findings and visualizations

## Skills Demonstrated

- Education policy analysis
- Excel-based data analysis
- Descriptive statistics
- Data visualization
- Civil Rights Data Collection interpretation
- Policy research communication
