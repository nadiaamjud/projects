# GIS Portfolio

This folder contains selected GIS and spatial analysis projects completed through coursework and applied policy research. The portfolio uses QGIS, GeoDa, Python, ACS/Census data, and spatial analysis methods to examine voting patterns, demographic inequality, urban development, segregation, and violence.

## Main File

- `spatial-analysis-portfolio.pdf`: Full GIS portfolio with project descriptions, methods, maps, and spatial skills demonstrated.

## Skills Demonstrated

- QGIS mapping and layout design
- Spatial joins and attribute joins
- Buffer analysis
- Choropleth mapping
- Coordinate reference system management
- Local Moran’s I / LISA cluster analysis
- GeoDa regression and residual mapping
- Python-based spatial analysis
- ACS/Census data integration
- Policy-oriented spatial interpretation

## Data and Project File Note

Final portfolio materials are included for easy viewing. Original spatial datasets and QGIS project files are not included because they may be large, course-provided, publicly available elsewhere, or dependent on local file paths.
